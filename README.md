# semicontinuous-ovl-paper-test
Code and synthetic data for the paper "Testing equality of two semicontinuous populations via overlapping coefficient method with application to dietary data". Includes implementation of the Overlapping Coefficient (OVL) test and dietary data analysis.
1) Algorithm Implementation: An algorithm for testing the equality of two semicontinuous populations based on the Overlapping     Coefficient (OVL).
2) Data Simulation: Scripts for generating synthetic data to validate the effectiveness of the proposed method.
3) Case Application: The data processing and analysis pipeline for the dietary data case study presented in the paper.
